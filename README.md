# MotoMate
Web app for tracking and managing vehicle costs.

## 🚧 Status
Project in progress – currently under active development.

## ✨ Features (planned)
- Add and manage vehicles
- Track expenses (fuel, service)
- Expense history
- Dashboard with summary

## 🛠 Tech stack
- React
- TypeScript
- Vite
- Redux Toolkit
- Tailwind CSS

## 🚀 Getting started

```bash
npm install
npm run dev
